## Run Locally

Clone the project

```bash
  git clone git@github.com:leeroynico/BadmintonCoachApp.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  expo start
```
